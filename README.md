This repository contains the initial application library developed for Veterans Development Group (VDG) and ForgeAI Solutions.

## Overview
This project demonstrates how AI-assisted development can be used to rapidly design, prototype, and deploy decision-support tools.

The first application in this library is an **Investment Analyzer App** designed to:
- Evaluate investment risk
- Assess job and income stability
- Generate actionable insights using AI-assisted logic

## Tech Stack
- AI-assisted development (Google AI Studio for initial prompting)
- Python / JavaScript (Replit-hosted)
- Modular design for future expansion
